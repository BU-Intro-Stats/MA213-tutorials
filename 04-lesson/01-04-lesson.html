Functions for Repeated Calculations

A function is useful when we need to do the same kind of calculation more than once. For example, probability from counts is:

\[ \text{probability} = \frac{\text{number of favorable outcomes}}{\text{number of possible outcomes}} \]

prob_from_counts <- function(favorable, total) {
  favorable / total
}

prob_from_counts(favorable = 2, total = 6)

Checking a Probability Distribution

A valid probability distribution has two features:

  • every probability is between 0 and 1;
  • all probabilities add up to 1.
outcomes <- c(0, 1, 2, 3)
probabilities <- c(0.10, 0.20, 0.40, 0.30)

sum(probabilities)
all(probabilities >= 0 & probabilities <= 1)

Expected Value

Expected value is a weighted average. Outcomes with larger probabilities count more heavily.

outcomes <- c(0, 1, 2, 3)
probabilities <- c(0.10, 0.20, 0.40, 0.30)

sum(outcomes * probabilities)

Simulating One Random Outcome

We can use sample() to draw one outcome according to a probability distribution.

set.seed(213)

sample(
  x = outcomes,
  size = 1,
  prob = probabilities
)

Repeating a Simulation

The function replicate() repeats the same expression many times.

set.seed(213)

simulated_outcomes <- replicate(
  n = 1000,
  expr = sample(outcomes, size = 1, prob = probabilities)
)

table(simulated_outcomes)

Simulated Probabilities

simulated_table <- tibble(outcome = simulated_outcomes) %>%
  count(outcome) %>%
  mutate(simulated_probability = n / sum(n))

simulated_table

Compare Simulation and Theory

theory_table <- tibble(
  outcome = outcomes,
  theoretical_probability = probabilities
)

left_join(theory_table, simulated_table, by = "outcome")
