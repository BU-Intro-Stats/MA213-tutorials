# MA213-tutorials

Interactive tutorials developed with the `learnr` package.

## Tutorial Sequence

| Tutorial | Folder | Focus |
| --- | --- | --- |
| 1 | `01-lesson` | R basics, data objects, vectors, and subsetting |
| 2 | `02-lesson` | Data import, cleaning, transformation, and summaries |
| 3 | `03-lesson` | Data visualization with `ggplot2` |
| 4 | `04-lesson` | Functions, repeated calculations, and probability simulation |
| 5 | `05-lesson` | Sampling distributions, confidence intervals, and p-values |
| 6 | `06-lesson` | Linear modeling workflow and interpretation |

Each lesson folder contains the source `.Rmd` tutorial and a rendered
`.html` file.
