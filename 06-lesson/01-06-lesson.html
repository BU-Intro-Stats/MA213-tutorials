Welcome

Hello, and welcome to Linear Modeling Workflow!

In this tutorial we will fit and interpret a simple linear model. The coding goal is to use lm() and supporting plots. The statistical goal is to explain what a model says in context and what it does not prove.

Our learning goals are to:

  • make an exploratory scatterplot before modeling;
  • fit a simple linear model with lm();
  • interpret the intercept and slope in context;
  • add a fitted line to a scatterplot;
  • inspect residuals and describe model limitations.

Packages

library(dplyr)
library(ggplot2)
library(tidyr)

Prepare the Data

We will use the built-in airquality data. To keep the modeling examples simple, we will remove rows missing Ozone, Temp, Wind, or Solar.R.

air_model_data <- as_tibble(airquality) %>%
  drop_na(Ozone, Temp, Wind, Solar.R)

glimpse(air_model_data)

Explore Before Modeling

Before fitting a model, plot the response and explanatory variable. We will model Ozone using Temp.

ggplot(air_model_data, aes(x = Temp, y = Ozone)) +
  geom_point() +
  labs(x = "Temperature", y = "Ozone")

Fit a Linear Model

The formula Ozone ~ Temp means:

response variable ~ explanatory variable
ozone_temp_model <- lm(Ozone ~ Temp, data = air_model_data)

summary(ozone_temp_model)

Model Coefficients

coef(ozone_temp_model)

The intercept is the predicted ozone level when temperature is 0 degrees Fahrenheit. The slope is the predicted change in ozone level for a 1-degree increase in temperature.

Predictions

Use predict() with a small data frame containing the explanatory variable values where we want predictions.

new_days <- tibble(Temp = c(70, 80, 90))

predict(ozone_temp_model, newdata = new_days)

Residuals

A residual is:

\[ \text{observed value} - \text{predicted value} \]

model_data_with_residuals <- air_model_data %>%
  mutate(
    predicted_ozone = predict(ozone_temp_model),
    residual = Ozone - predicted_ozone
  )

head(model_data_with_residuals)

Residual Plot

ggplot(model_data_with_residuals, aes(x = predicted_ozone, y = residual)) +
  geom_point() +
  geom_hline(yintercept = 0) +
  labs(x = "Predicted ozone", y = "Residual")
